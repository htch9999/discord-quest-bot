Bảo mật & Minh bạch

Chúng tôi công khai mọi thứ.

  • Token mã hoá bằng AES-256-GCM
  • Key dẫn xuất riêng biệt từng Discord user ID
  • Token không bao giờ xuất hiện trong log
  • Mã nguồn 100% public trên GitHub
  • Hướng dẫn tự host có sẵn
  • Xoá token bất cứ lúc với /autoquests remove
Code mã hoá: Xem GitHub 
def encrypt_token(token: str, uid: str) -> bytes:
    salt = os.urandom(16)
    key = PBKDF2HMAC(
        algorithm=hashes.SHA256(),
        length=32,
        salt=salt + uid.encode(),
        iterations=260_000,
    ).derive(MASTER_SECRET.encode())
    nonce = os.urandom(12)
    cipher = AESGCM(key)
    ct = cipher.encrypt(nonce, token.encode(), None)
    return base64.b64encode(salt + nonce + ct)

Danh sách lệnh

Đơn giản, mạnh mẽ, riêng tư.

Lệnh Mô tả Ví dụ
/quests {token} Chạy một lần, không lưu token /quests MTA...
Lệnh Mô tả Ví dụ
/autoquests {token} [label] Lưu token + tự động mỗi 2 ngày /autoquests MTA... main
/autoquests run [label] Chạy thủ công ngay /autoquests run main
/autoquests pause [label] Tạm dừng auto /autoquests pause main
/autoquests resume [label] Bật lại auto /autoquests resume main
Lệnh Mô tả Ví dụ
/autoquests list Danh sách token đã lưu /autoquests list
/autoquests remove {label} Xoá token + thống kê /autoquests remove main
/autoquests rename {old} {new} Đổi tên hiển thị /autoquests rename main alt
Lệnh Mô tả Ví dụ
/autoquests status [label] Xem trạng thái token /autoquests status main
/autoquests-info Thống kê cá nhân /autoquests-info
/info Thông tin server bot /info
/help Hướng dẫn sử dụng /help

Cách lấy Discord Token

Lấy token an toàn từ trình duyệt của bạn

Phương pháp mới (Khuyên dùng)

  • Mở Developer Tools bằng phím F12 hoặc Ctrl + Shift + I.
  • Bật chế độ giả lập thiết bị di động (Mobile Device Emulation) bằng Ctrl + Shift + M.
  • Dán đoạn code sau vào tab Console và nhấn Enter:
Console Script 
const iframe = document.createElement('iframe');
console.log(
  'Token: %c%s',
  'font-size:16px;',
  JSON.parse(document.body.appendChild(iframe).contentWindow.localStorage.token)
);
iframe.remove();

* Hoặc mở tab Application > Local Storage và tìm key token dưới tên miền discord.com (khi đang bật giả lập mobile).

Phương pháp cũ (Ấn để mở rộng)
  • Nhấn F12 hoặc Ctrl + Shift + I để mở Developer Tools.
  • Chuyển sang tab Network và lọc Fetch/XHR.
  • Bấm vào một channel bất kì để thấy Request mới xuất hiện.
  • Kiểm tra mục Request Headers > Authorization để copy token của bạn.
