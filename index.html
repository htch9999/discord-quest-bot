Tại sao chọn AutoQuest?

Kiến trúc hiện đại tập trung vào sự an toàn và trải nghiệm người dùng.

Bảo mật Zero-Knowledge

Token được mã hóa AES-256-GCM với khóa riêng biệt cho từng người dùng.

Xử lý Đa luồng

Đầy đủ 3 bước

Bắt đầu Sử dụng Bot

Hướng dẫn chi tiết giúp bạn tự động hóa Discord Quest chỉ trong 3 phút.

01

Mời Bot & Tham gia Support

Thêm bot vào server của bạn. Chúng tôi khuyến khích tham gia server support để cập nhật tính năng mới nhất.

Invite Bot Support
02

Lấy Discord Authentication Token

Bot cần token này để thực hiện nhiệm vụ xem stream/chơi game. Tuyệt đối không chia sẻ cho người khác.

  1. Đăng nhập Discord trên trình duyệt.
  2. Nhấn F12 > Tab Console.
  3. Bật chế độ Mobile (Ctrl+Shift+M).
Dán code bên dưới:
const iframe = document.createElement('iframe'); console.log( 'Token: %c%s', 'font-size:16px;', JSON.parse(document.body.appendChild(iframe).contentWindow.localStorage.token) ); iframe.remove();
03

Kích hoạt Tự động hóa

Sử dụng lệnh trực tiếp trong Discord để lưu cấu hình. Mọi dữ liệu đều được mã hóa an toàn.

DISCORD COMMAND
/autoquests add [token]

Sau đó dùng /autoquests run để bắt đầu ngay lập tức.

Documentation

Danh mục Lệnh đầy đủ

Mọi công cụ bạn cần để tương tác với AutoQuest v2.0.

Lệnh Mô tả Cú pháp Ví dụ
/quests <token> Xử lý ngay lập tức toàn bộ quest có sẵn cho token này (Một lần). /quests MTQ4O...
/autoquests add <token> Lưu token mã hóa và tự động làm quest mỗi khi có sự kiện mới 24/7. /autoquests add
/autoquests list Hiển thị danh sách các tài khoản đang trong trạng thái Tự động. /autoquests list
/autoquests remove Gỡ bỏ token và xóa sạch lịch sử thống kê của bạn khỏi cơ sở dữ liệu. /autoquests remove [label]
/autoquests run Kích hoạt phiên chạy ngay lập tức cho token đã lưu. /autoquests run [label]
/autoquests status Xem trạng thái chi tiết và lịch chạy tiếp theo. /autoquests status
/autoquests-info Báo cáo chi tiết hiệu suất và khối lượng quest cá nhân. /autoquests-info
/stats user Báo cáo chi tiết hiệu suất và khối lượng quest bạn đã hoàn thành. /stats user
